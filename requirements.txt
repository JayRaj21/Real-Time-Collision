# =============================================================================
# requirements.txt — Jetson Orin Nano (JetPack 5.x / 6.x · L4T aarch64)
# =============================================================================
#
# IMPORTANT — READ BEFORE pip install -r requirements.txt
# ─────────────────────────────────────────────────────────────────────────────
#
# 1. PyTorch  ──  DO NOT install from PyPI.
#    The PyPI wheel is built for x86-64 and will either fail to install or run
#    without CUDA on aarch64.  Install the NVIDIA-provided Jetson wheel instead.
#    See README.md §2 for the exact download URL and install command.
#
# 2. OpenCV   ──  DO NOT install opencv-python from PyPI.
#    JetPack ships a GStreamer-enabled OpenCV build (python3-opencv).
#    The PyPI wheel lacks the nvarguscamerasrc GStreamer plugin support and
#    will break the CSI camera pipeline.
#    Install via: sudo apt install python3-opencv
#
# Install everything else with:
#    pip install -r requirements.txt
# =============================================================================
